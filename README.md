### 简单易用的AI桌面工具

#### 1. 图像生成
- 文生图：输入提示词，生成图片（仅支持英文）
- 图生图：根据图片及提示词生成图片
- 图像无损放大：比如将 512*512 放大到 1024 * 1024
- 分辨率 512*512 25步 CPU(i5处理器) 5分钟。 3060显卡20秒。
